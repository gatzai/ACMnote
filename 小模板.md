# 快速幂
```C++
//迭代
int pow2(const int a,const int b){
    int r = 1, base = b;
    while(b!=0){
        if(b%2 == 0)r*=base;
        base*=base;
        b/=2;
    }
    return r;
}
```

```C++
//递归
int f(int m,int n){ 
    if(n==0) return 1;
    if(n==1) return m;
    int temp=f(m,n/2);
    return (n%2==0 ? 1 : m)*temp*temp;
}
```

#GCD
```C++
int gcd(int a, int b){
    return b>0?gcd(b,a%b):a;
}
```
