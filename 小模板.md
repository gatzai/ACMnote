# 快速幂
```C++
//迭代
int pow2(const int a,int b){
    int r=1, base = a;
    while(b!=0){
        if(b&1!=0)r*=base;
        base*=base;
        b>>=1;
    }
    return r;
}
```

```C++
//递归
int f(int m,int n){ 
    if(n==0) return 1;
    if(n==1) return m;
    int temp=f(m,n/2);
    return (n%2==0 ? 1 : m)*temp*temp;
}
```

# GCD(Great Common Divisor)
```C++
int gcd(int a, int b){
    return b>0?gcd(b,a%b):a;
}
```
