# 快速幂
```C++
//迭代
int pow2(const int a,int b){
    int r=1, base = a;
    while(b!=0){
        if(b&1!=0)r*=base;
        base*=base;
        b>>=1;
    }
    return r;
}
/*
比如：11的二进制是1011，11 = 2³×1 + 2²×0 + 2¹×1 + 2º×1，因此，我们将a¹¹转化为算 a2^0*a2^1*a2^3，也就是a1*a2*a8，
看出来快的多了吧！原来算11次，现在算三次。
*/
```

```C++
int f(int m,int n){ 
    if(n==0) return 1;
    if(n==1) return m;
    int temp=f(m,n/2);
    return (n%2==0 ? 1 : m)*temp*temp;
}
```

# GCD(Great Common Divisor)
```C++
int gcd(int a, int b){
    return b>0?gcd(b,a%b):a;
}
```
